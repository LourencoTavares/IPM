# Bake-off #1 — Rede Social de Receitas (IPM)

Projeto desenvolvido no âmbito da UC de **Interação Pessoa-Máquina (2025/2026)**.

---

## Descrição

Este projeto consiste no **desenho da interface de utilizador (UI)** de uma aplicação móvel para uma rede social de partilha de vídeos de receitas.

O objetivo foi criar um **protótipo interativo de alta-fidelidade (PAF)**, desenvolvido em Figma, que permita aos utilizadores:
- pesquisar receitas com base em filtros avançados  
- interagir com conteúdos gastronómicos  
- explorar sugestões com base em ingredientes da época  

O foco do projeto foi na **experiência do utilizador (UX)** e não na implementação funcional do sistema.  

---

## Objetivo

Desenvolver uma interface:
- intuitiva  
- rápida  
- apelativa  
- orientada às necessidades do utilizador  

seguindo um processo iterativo de:
> ideação -> prototipagem -> avaliação

---

## Funcionalidades Implementadas

O protótipo suporta as seguintes funcionalidades principais:

### 1. Gestão de utensílios
- adicionar utensílios ao perfil do utilizador  
- exemplo: wok, batedeira elétrica, termómetro culinário  

### 2. Pesquisa de receitas
- filtros por:
  - tipo de cozinha (ex: asiática)  
  - ingredientes (ex: tofu)  
  - restrições alimentares (ex: vegetariano)  
  - utensílios necessários  
- possibilidade de subscrever criadores  

### 3. Inspiração com ingredientes da época
- notificações de ingredientes sazonais  
- sugestões automáticas de receitas  
- localização de mercados com ingredientes disponíveis  

---

## Processo de Desenvolvimento

O projeto seguiu um processo iterativo:

### Protótipo de Baixa-Fidelidade (PBF)
- desenvolvido com papel e materiais físicos  
- foco na estrutura e fluxo da interface  

### Avaliação Formativa
- testes com utilizadores  
- métodos utilizados:
  - Think-aloud  
  - Wizard-of-Oz  

### Protótipo de Alta-Fidelidade (PAF)
- desenvolvido em **Figma**  
- interface final interativa  
- otimização com base nos resultados da avaliação
